L11: Interface Specifications — Digital I/O Cards

Source: DeltaV Scalable Process System — Hardware Installation and Reference Manual, D800001X312, April 2021 (Emerson). Appendix C, "Interface Specifications," pp. 125–178.

This lesson provides a comprehensive master reference for every digital (discrete) I/O card supported by the DeltaV system. Use the tables below to select the correct card for your application, verify wiring compatibility, and plan your power budget.

11.1 — Digital I/O Card Types at a Glance

Key Concept: DeltaV digital I/O cards fall into three categories: Discrete Input (DI) for reading on/off signals, Discrete Output (DO) for driving on/off loads, and Discrete Shared Control (DSC) for specialized applications. Each card type is available in multiple voltage ranges (24 VDC, 120 VAC, 230 VAC), isolation configurations (dry contact vs. isolated), and channel counts (8 or 32 channels). Series 2 versions add wider temperature ranges and redundancy support.

Pitfall — 32-Channel DO Fuse Groups: A resettable 1A fuse protects each group of four channels (1–4, 5–8, … 29–32). Excessive current on any single channel can trip the fuse and disable all four channels in the group. Turn off or disconnect the group and allow the fuse to cool and reset.

11.5 — Configurable Channel Types (DO Cards)

DeltaV DO cards support three configurable channel types. Each channel can be independently configured:

Channel Type Description Fail-Action: Hold Last Value Fail-Action: Go to Configured State
Discrete Output Output stays in the last state submitted by the controller Output holds last state Output drives to configured faultstate value
Momentary Output Output is active for a pre-configured time period (100 ms to 100 s) Output finishes current pulse Output drives to faultstate; channel re-configured as latched
Continuous Pulse Output Output is active as a percentage of a pre-configured base time period (100 ms to 100 s) Output continues pulsing Output drives to faultstate; channel re-configured as latched

11.6 — Dry Contact vs. Isolated: When to Use Each

Key Concept:
• Dry Contact cards share a common return path. They are simpler and less expensive but require that all connected devices reference the same voltage potential.
• Isolated cards provide channel-to-channel isolation (tested to 1500 VDC for DC cards, 250 VAC for AC cards). Use isolated cards when field devices operate at different voltage potentials, when you need to prevent ground loops, or when field wiring runs through different areas with different ground references.
